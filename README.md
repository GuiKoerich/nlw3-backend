# Next Level Week - 3

Projeto do backend escrito com TypeScript em NodeJS.

Projeto desenvolvido pelo curso da [Rocketseat](https://rocketseat.com.br)


### Instalação
---

Após baixar o projeto execute:
```
yarn install
```

Após a instalação execute o seguinte comando para criação do banco de dados:
```
yarn orm migration:run
```

Depois de finalizada as migrations, para rodar o projeto em modo de desenvolvimento:
```
yarn dev
```
